# NHS Waiting Times Pipeline

## Objective

This project demonstrates a data engineering workflow for ingesting, validating, transforming and loading NHS waiting times data.

## Architecture

CSV Source
↓
Python ETL
↓
Validation
↓
SQLite Database
↓
Reporting Layer
